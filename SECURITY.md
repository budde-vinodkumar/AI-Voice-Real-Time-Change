# Security Policy

## Responsible Disclosure
If you discover a vulnerability in this project, we ask that you provide us with a reasonable amount of time to address it before disclosing it publicly. Please follow these guidelines:

1. **Do Not Share**: Do not disclose any information about the vulnerability until it has been resolved.
2. **Contact Us**: Please send a detailed report of the vulnerability to the following email: 
   [security@ai-voice.org](mailto:security@ai-voice.org)
3. **Provide Context**: Include information such as how to reproduce the issue, potential impact, and any related security information.

## Contact Information
For any security-related inquiries, please contact:

- **Email**: [security@ai-voice.org](mailto:security@ai-voice.org) 

Thank you for helping to keep this project secure!
